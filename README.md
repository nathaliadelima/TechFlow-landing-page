# TechFlow Landing Page

Este projeto é uma landing page desenvolvida para apresentar o TechFlow, um template criado no Notion com foco na organização de estudos para estudantes de programação. A proposta da página foi criar um espaço moderno, organizado e visualmente atrativo para divulgar o template, destacando suas funcionalidades, benefícios e proposta de valor.

O projeto foi desenvolvido durante meus estudos em programação Front-End, com o objetivo de praticar construção de interfaces, responsividade e estruturação de páginas de vendas. Além da parte visual, também trabalhei na experiência do usuário, buscando deixar a navegação simples, clara e agradável.

Esse projeto representa mais uma etapa importante da minha evolução como desenvolvedora, permitindo unir programação, design e criatividade em um projeto voltado para produtividade e organização nos estudos.

## Objetivo

O principal objetivo foi praticar o desenvolvimento de uma landing page utilizando HTML, CSS e JavaScript, simulando uma página real de apresentação e divulgação de produto digital.

Também busquei melhorar minhas habilidades na criação de layouts modernos, organização visual de conteúdo e desenvolvimento de interfaces responsivas.

## Funcionalidades

* Apresentação do template TechFlow
* Seções explicando funcionalidades e benefícios
* Layout moderno e organizado
* Design responsivo para diferentes dispositivos
* Navegação simples e intuitiva
* Interações utilizando JavaScript

## Tecnologias utilizadas

* HTML5 semântico
* CSS3
* JavaScript

## Estrutura

* `index.html` — estrutura da página
* `styles.css` — estilos
* `script.js` — interações
